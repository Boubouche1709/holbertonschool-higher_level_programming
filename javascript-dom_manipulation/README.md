## üß† Project Content (Examples)

- **Select DOM elements** using `getElementById()`, `querySelector()`, etc.  
- **Modify element content** with `innerText` or `innerHTML`  
- **Update element styles** with `element.style.property`  
- **Add/remove CSS classes** using `classList.add()` / `classList.remove()`  
- **Listen to events** (`click`, `submit`, `keydown`, etc.) using `addEventListener()`  
- **Create and insert elements** dynamically using `createElement()` and `appendChild()`  
- **Perform network requests** and display data without reloading the page
